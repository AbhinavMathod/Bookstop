### Steps to run
1. Clone repository.
2. Navigate to server directory
3. Run `node server.js`
4. Navigate to webapp directory
5. Run `npm install`
6. Run `ng build`
7. Run `ng serve`
8. This will navigate to `http://localhost:4200/`
